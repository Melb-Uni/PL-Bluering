## Local Dev 
- Required Python version 3.6

     ```
     cd prototypes\website_demo
     pip install -r requirements.txt
     python app.py
     ```

## Docker Build
- Install Docker
- start Docker
    ```
    docker compose up
    ```
- Test the web server is running by visiting `localhost:5000` in the browser.
